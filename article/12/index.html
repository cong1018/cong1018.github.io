1.下载

1.github下载地址

2.安装

点击下载的软件直接安装，默认安装在c盘Program Files (x86)中，可以选择是否剪切到其他盘，不更换盘符后续安装的node文件全在c盘；

5.npm报错处理

以下是npm -v在运行出现时 npm WARN config global --global, -local are deprecated. Use --location=global instead.解决步骤

步骤一：打开node.js的安装路径–>找到npm.cmd文件，右键编辑打开

步骤二：将文件中的prefix-g 更改为prefix –location=global。如同所示。注意后面还有个单引号！！！不要误删！！！

6.配置npm连接私服

因开发人员不允许上外网，故在10.88.12.5服务器上搭建npm私服，开发人员本地需将npm配置为连接私服地址，方法如下：
1、win+R快捷键打开“运行”，输入cmd，进入Windows命令终端；
2、输入命令：npm set registry http://10.88.12.185:7001/
3、按Enter键确认即可
4、查看私服仓库详情可以访问 http://10.88.12.185:7002/
